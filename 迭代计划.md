## v0.02 核心改进

### 数据 — 全量训练
- [ ] IterableDataset 流式读取 5839 只股票，解决 8GB 内存瓶颈 (54万样本 → 1500万)
- [ ] 每轮逐文件读取→计算特征→喂给模型，用完即弃

### 特征 — 独立信号源
- [ ] **大盘语境**：上证/沪深300当日涨跌幅（让模型知道牛熊）
- [ ] **市场情绪**：全市场涨停家数、连板高度、炸板率
- [ ] **板块效应**：所属板块当日涨幅、板块涨停家数
- [ ] **连板记忆**：过去 5/10/20 日涨停次数的衰减加权和
- [ ] **竞价信号**：开盘高开幅度（开盘/昨收 - 1）
- [ ] **全市场排名分位**：每只特征在全市场的排名百分位

### 模型 — 多目标 + 训练策略
- [ ] 多任务输出头：同时预测 (次日收益/3日/5日收益/5日最高/涨停flag)
- [ ] 分级训练：全量股票预训练 → 候选池微调
- [ ] 更强的正则化：增加 dropout、label smoothing
- [ ] 特征重要性分析（基于 Attention 权重）

### 架构升级
- [ ] 每日流水线接入模型预测（scaler 已保存）
- [ ] 断点续训（保存 optimizer + scheduler + epoch state）
